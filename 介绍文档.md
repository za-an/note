# 智慧伴学系统

## 产品介绍文档

一、产品创意背景

高校线下课堂存在诸多学习痛点：课后复习资料稀缺，授课内容无结构化整理，优质笔记难以班级流通；通用 AI 脱离本节课内容，辅助效果低下。现有学习软件多为跨平台工具，课程针对性极低，文件传输依赖云端，UI 交互简陋。市面上缺少适配鸿蒙生态、针对课堂场景的 AI 协同学习工具，本产品针对空白赛道，融合华为鸿蒙互联等技术与课堂 AI 知识库方案，解决师生学习难题。

**二、产品创意内容**

产品核心创意三点：1.深度结合华为特色技术，实现 NFC 碰一碰互联、“隔空抓取”文件、跨设备学习流转，搭配精美动画 UI 展示效果；2.搭建完整“学-复习”闭环，课堂录音转写后通过 RAG 生成课程知识库、Markdown 提纲；3.构建师生共创资源池，教师审核官方提纲，学生笔记可选择共享，AI 评估优质内容反向扩充知识库，形成班级学习资源体系。

**三、产品实现路径**

基于鸿蒙 ArkTS 开发，分层落地： 1. 采集层：调用系统音频接口，实时语音转文字； 2.AI层：RAG 模型分割文本、提取知识点，生成 Markdown 提纲； 3. 业务层：实现教师审核发布、AI 问答、内容质量评估； 4. 鸿蒙协同层：接入超级终端，开发 NFC 配对、文件互抓、跨端学习接续。

**四、核心功能设计**

1. 全程录音转写，AI 自动生成结构化提纲，教师确认后上传班级共享池； 2. 构建仅依托本课程 RAG 知识库作答的AI，回复贴合课堂授课内容； 3. 支持图文笔记编辑，分隐私、共享模式，优质笔记扩充资源池； 4. 支持NFC 快速配对、局域网拖拽抓取文件、跨设备无缝接续学习进度； 5. 提供知识点热力图、错题归集、MD/PDF 多格式导出等其他功能。

**五、市场前景评估**

目标用户为全国鸿蒙设备高校师生，用户基数大、学习需求强。同类产品缺少鸿蒙独有硬件协同能力，同质化严重，本项目依靠 NFC 本地传输、端侧 RAG 知识库形成竞争优势。产品轻量无广告，适配课堂、自习、宿舍多场景，可面向校内小规模试点推广，后续可拓展多校校园合作；伴随鸿蒙设备在高校普及，设备协同类学习工具需求持续上涨，产品具备落地价值与长期拓展空间。
